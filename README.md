# 2D-Arcade-Game
Game about wanting kisses and being hunted by pugs, in a generic 2D format. 

## How to get executable


```
mkdir pug_game.app
```

Download `A Build File/Kissie 2D Game.app/contents.zip`

Extract in folder `pug_game.app`

If on MacOs, this should run my application.
